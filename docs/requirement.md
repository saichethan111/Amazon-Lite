Customer Capabilities:
A Customer can:
    -> View Products
    -> View Product Details
    -> Place Order
    -> View Order

Admin Capabilities:
An Admin can:
    -> Create Product
    -> Update Product
    -> Add Inventory

Out of scope for this release at least:
    -> Payment Processing
    -> Shipping
    -> Notifications
    -> Authentication
    -> Kafka
    -> Redis
    -> Kubernetes
