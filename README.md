# Amazon-Lite

Amazon-Lite is a commerce platform learning project. It starts as a Spring Boot and PostgreSQL modular monolith, then gradually evolves into microservices, Kafka, Redis, Docker, Kubernetes, observability, and production hardening.

## Current Phase

Sprint 0 / Week 1 focuses on the foundation:

- Requirements
- Domain model
- Database design
- API design
- Modular monolith implementation

Kafka, Redis, Kubernetes, authentication, payment, shipping, and notifications are intentionally out of scope for the MVP.
